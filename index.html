캐릭터 생성 가이드라인

새로운 캐릭터를 설명할 때 다음 요소를 포함하면 좋습니다. 프롬프트가 자세하고 구체적일수록 원하는 결과에 더 가까워집니다.

핵심 아이디어를 먼저 설정하고, 세부 사항을 추가하며 반복적으로 개선하는 것이 중요합니다.

1. 주제 (Subject):

  • 캐릭터의 핵심 대상(인물, 동물, 사물 등)을 명확히 정의합니다.

2. 배경 및 맥락 (Context & Background):

  • 캐릭터가 놓일 환경이나 상황을 설명합니다. (예: 실내 스튜디오, 야외 풍경, 특정 시대의 배경 등)

3. 스타일 (Style):

  • 원하는 이미지의 시각적 스타일을 지정합니다. (예: 회화, 사진, 스케치, 디지털 아트, 특정 예술 사조 등)
  • 선의 두께, 색상 팔레트, 전반적인 분위기 등 구체적인 스타일 요소를 추가할 수 있습니다.

4. 상세 묘사 및 품질:

  • 자세한 형용사와 부사를 사용하여 캐릭터를 명확하게 묘사합니다.
  • 필요하다면 특정 아티스트나 예술 사조를 언급하여 원하는 미학을 표현할 수 있습니다.
  • 이미지 품질을 높이려면 '고품질(high-quality)', '아름다운(beautiful)', '세밀한(detailed)', '4K', 'HDR'과 같은 키워드를 사용할 수 있습니다.

Gemini API 키 발급 가이드

Gemini API 키는 Google AI Studio에서 발급받을 수 있습니다. 다음 단계를 따르세요:

  1. Google AI Studio API 키 페이지로 이동합니다.
  2. Google 계정으로 로그인합니다.
  3. "API 키 만들기" 또는 "API 키 가져오기" 버튼을 클릭합니다.
  4. 생성된 API 키를 복사하여 이 페이지의 "Gemini API 키" 입력란에 붙여넣습니다.
  5. 참고: 이미지 생성 기능을 사용하려면 Google Cloud 프로젝트에 유효한 결제 계정이 연결되어 있어야 하며, Vertex AI API가 활성화되어 있어야 합니다.

API 키는 민감한 정보이므로, 외부에 노출되지 않도록 주의하세요.
